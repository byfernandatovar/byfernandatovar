# Portfolio Images

## Structure

Add your portfolio images in the following folders:

- `/weddings/` - Wedding photography
- `/portraits/` - Portrait photography  
- `/moments/` - Life moments photography
- `/couples/` - Couples photography

## Naming Convention

Images should be named numerically starting from 1:
- `1.jpg` or `1.webp`
- `2.jpg` or `2.webp`
- `3.jpg` or `3.webp`
- etc.

The gallery will automatically load up to 12 images per category.

## Main Category Images

For the main portfolio page cards, use `1.jpg` from each category as the cover image.

## Supported Formats

- `.jpg` / `.jpeg`
- `.webp`
- `.png`

## Recommended Sizes

- Vertical images: 1200px × 1600px (3:4 ratio)
- Horizontal images: 1600px × 1200px (4:3 ratio)
- Square images: 1200px × 1200px (1:1 ratio)

## Notes

- Images can be mixed (vertical and horizontal) for a more dynamic gallery
- Optimize images for web (recommended max 500KB per image)
- Use WebP format for better performance
